# PR #68 - Angular Gateway environment config

## Files changed

- `frontend/angular-app/src/environments/environment.ts`
- `frontend/angular-app/src/app/gateway-api.service.ts`
- `frontend/angular-app/src/app/gateway-api.service.spec.ts`
- `frontend/angular-app/README.md`

## Changes

- Adds Angular environment configuration for Gateway routing.
- Moves the advisor dashboard path out of `GatewayApiService` hard-coded state.
- Uses `gatewayBaseUrl` and `advisorDashboardPath` to build the advisor dashboard URL.
- Adds service test coverage for the configured endpoint.
- Updates the Angular README with Gateway configuration details.

## Behavior

```text
- default Gateway base URL is empty
- default advisor dashboard path is /api/advisor/dashboard
- service calls the environment-configured advisor dashboard endpoint
```

## Impact

- No backend runtime behavior change.
- Angular Gateway URL is now configurable from environment state.
- Prepares future environment-specific Gateway URLs.
