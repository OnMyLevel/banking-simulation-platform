# PR #69 - Angular local Gateway proxy

## Files changed

- `frontend/angular-app/proxy.conf.json`
- `frontend/angular-app/package.json`
- `frontend/angular-app/README.md`

## Changes

- Adds a local Angular proxy configuration for Gateway API calls.
- Adds `npm run dev:proxy` to run Angular with the proxy enabled.
- Documents the local Gateway proxy setup in the Angular README.

## Behavior

```text
- Angular /api requests are forwarded to http://localhost:8080
- standard npm run dev remains unchanged
- npm run dev:proxy enables local Gateway integration
```

## Impact

- No backend runtime behavior change.
- No production Angular build behavior change.
- Improves local development against the Gateway service.
